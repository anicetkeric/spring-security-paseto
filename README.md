# spring-security-paseto

Secure REST API with Spring Boot 3.0, Spring Security 6.0 and PASETO

## Tech Stack

**Spring-boot**, **maven**, **java17**, **PostgreSQL**, **PASETO**

## Run Locally

Clone the project

```bash
  git clone https://github.com/anicetkeric/spring-security-paseto.git
```
